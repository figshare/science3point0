The BackType Tweetcount plugin shows the number of tweets your posts get and allows users to retweet.

== Description ==

The BackType Tweetcount plugin lets you show the number of times each of your blog posts have been linked to on Twitter. It also lets users easily and quickly retweet a link to your post.

The plugin is highly customizable. You can specify:

*	Which of the following URL shorteners to use for retweets: [bt.io](http://bt.io) (default), [awe.sm](http://awe.sm) (custom), [bit.ly](http://bit.ly/), [su.pr](http://su.pr/), [digg.com](http://digg.com/), [tinyurl](http://tinyurl.com/)
*	What text you would like used for the retweets
*   The color, size and location of the button

This plugin was created using [BackType Connect](http://www.backtype.com/connect).

== Installation ==

1. Download the plugin
2. Unzip the plugin and upload the `backtype-tweetcount` directory to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure the plugin through the 'Settings' &gt; 'BackType Tweetcount' menu option

== Frequently Asked Questions ==

= How does manual positioning work? =

To use manual positioning, `echo backtype_tweetcount();` where you would like the plugin to appear in your WordPress theme.

= How long do you track tweets for? =

Currently, we will track tweets for your posts for up to approximately three months (much longer than other plugins)

= How can I use a URL shortener other than the ones listed? =

Add a custom field to your post named `bttc_short_url` and enter the short URL you would like to use.
